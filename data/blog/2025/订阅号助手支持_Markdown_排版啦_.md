今天要给大家分享一个超棒的消息呀 🎉 订阅号助手现在支持 Markdown 排版咯 👏

最近 mdxnotes 也有了新的更新，它可是全网唯一一个支持移动端访问的 Markdown 排版编辑器哦 👍 我们只需要把从 ChatGPT 或者 deepseek 获取到的文本内容，直接粘贴到 mdxnotes 里，然后就可以一键拷贝到订阅号助手，是不是超级方便呀 🤩

而且呀，mdxnotes 还扩展了 Markdown 语法呢 👇  
用两个 == 就可以实现文本高亮哦，在 ^ 之间添加下划线 ，是不是很有用呀。 有了这些新功能，我们在订阅号上排版文章就更加轻松高效啦，能够让我们的文章更加美观、有条理，吸引更多的读者哦 💕 大家赶紧去试试吧。
