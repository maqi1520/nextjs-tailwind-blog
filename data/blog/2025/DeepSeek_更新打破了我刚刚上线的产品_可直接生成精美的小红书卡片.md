最近，我上线一款应用 md2crad 可以将 markdown 转变为知识卡片， 但是昨天 DeepSeek 的这次更新， 让我刚上线的产品就失去了价值。因为可以直接在 deepseek 中实现知识卡片的制作，并且 UI 审美也非常出色。

下面我来演示下步骤

首先我让 DeepSeek 写一段小红书知识卡片的内容。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/e4YNLngAJ87OvaVYibZdKPibaia7picOEdo7R3r6JJ2Hxh4SESDscHfPo5pMYhUhgibiaX6CVicDkDXZCQn8NSKCehOdw/640?wx_fmt=png&from=appmsg)有了内容我就可以生成知识卡片了，下面是我的提示词

```text
请使用 html 和 css 实现一张小红书知识卡片，设计风格是

 酸性设计（Acid Design）

- 🎸暗黑系+金属质感+故障艺术

- 🖥️适合：音乐/潮牌/实验性网站

- 💡要点：用镭射渐变+锯齿字体

卡片内容：

<content>
# 💻【2024设计师必看！这8种网页设计风格太出片了】💻

![首图](https://picsum.photos/600/300)

1. **玻璃拟态2.0（Glassmorphism）**
   - ✨特点：毛玻璃效果+动态光影
   - 🔍案例：苹果官网导航栏
   - 📱适配：科技类/元宇宙主题APP

2. **酸性设计（Acid Design）**
   - 🎸暗黑系+金属质感+故障艺术
   - 🖥️适合：音乐/潮牌/实验性网站
   - 💡要点：用镭射渐变+锯齿字体

3. **Web3.0风格**
   - 🔗区块链元素+像素风+霓虹灯效
   - 🌐案例：NFT交易平台UI
   - 💻设计工具：Figma社区Web3组件库

4. **新丑风（Brutalism）**
   - �故意"粗糙"的排版+高饱和撞色
   - 📌出圈原因：反精致化设计趋势
   - ⚠️警告：甲方接受度两极分化

5. **Y2K千禧风复兴**
   - 📟低多边形+糖果色+老电脑元素
   - 🎀最近爆火：韩国小众品牌官网
   - 🛒带货点：适合美妆/复古穿搭类电商
```
