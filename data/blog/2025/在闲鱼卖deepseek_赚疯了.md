打开闲鱼，随便看了下，销量还挺高，先有在闲鱼卖部署教程，后有卖 APIkey。

目前，deepseek 官方声明称，“当前服务器资源紧张，为避免对您造成业务影响，已暂停 API 服务充值。"

分享几个我知道的大模型 api 服务。

硅基流动:https://siliconflow.cn/zh-cn/  
DMXAPI:https://dmxapi.com/  
火山引擎（目前最便宜，官方价一半）:https://volcengine.com/  
openrouter：https://openrouter.ai/deepseek/deepseek-r1

按官方 deepseek-reasoner 模型上线即按每百万输入 tokens 4 元，每百万输出 tokens 16 元计费。卖一个能赚 30。

💡 赚钱的机会一直都在，只是我们没抓住信息差！ 💡
