我使用的是 GitHub copilot

GitHub Copilot Free（免费版）  
Copilot Chat 每月总共限制 50 条聊天消息。

GPT 4o (默认模型)  
这是 Copilot Chat 的默认模型  
是一个多功能的多模态模型，在文本和图像处理方面都表现出色  
设计用于提供快速、可靠的响应  
在非英语语言方面表现优秀

Claude 3.5 Sonnet  
擅长整个软件开发生命周期的编码任务  
涵盖从初始设计到 bug 修复、维护到优化等各个方面  
托管在 Amazon Web Services 上

Gemini 2.0 Flash  
具有强大的编码、数学和推理能力  
特别适合协助软件开发

o1  
专注于高级推理和解决复杂问题，特别是在数学和科学领域  
响应速度比 GPT 4o 模型慢  
每天限制使用 10 次请求

o3-mini  
这是推理模型的下一代版本  
在编码基准测试方面优于 o1  
响应时间与 o1-mini 相当  
最适合代码生成和小上下文操作  
每 12 小时限制 50 次请求
